# Skill Factory

This repository mirrors the current Codex global skill library from `~/.codex/skills`.

What is included:
- all Codex skill source directories
- manifests, references, scripts, assets, and agent metadata
- hidden system skills under `.system/`

What is intentionally excluded:
- generated reports
- wisdom ledgers
- transient `_generated` state
- backup directories and Python cache files

The goal is to keep the GitHub copy focused on portable, reusable skill source rather than local runtime residue.
