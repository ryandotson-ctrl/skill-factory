# Apple Project Bootstrap Guidance

- Use `$apple-project-bootstrap-orchestrator` for future scaffold or adoption refreshes.
- Use `$xcode-build-harness-installer` when the local Apple CLI harness needs installation or upgrade.
- Use `$qa-automation-engineer` before sign-off on app-facing regressions.
- Use `$apple-ecosystem-release-operator` when the project is ready for archive, TestFlight, or release evidence capture.
