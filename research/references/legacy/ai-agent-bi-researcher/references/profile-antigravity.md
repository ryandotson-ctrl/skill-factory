# Optional Profile: Antigravity

Load this file only when Antigravity workspace context is active.

## Profile Objective
Translate generic trend findings into practical changes for a file-based, skill-oriented agent ecosystem.

## Adaptation Rules
1. Map each top trend to:
   - an existing skill upgrade, or
   - a new skill proposal.
2. Prefer additive changes over rewrites.
3. Keep all recommendations path-agnostic and portable.

## Output Additions
- `Antigravity Skill Impact`: list affected skills and expected lift.
- `Routing Considerations`: event or dispatch implications.
- `Adoption Sequencing`: low-risk rollout order.

## Risk Focus
- Skill overlap and duplicate behavior.
- Event routing noise and dispatch loops.
- Portability regressions from host-specific assumptions.
