# Worked Examples

## Example 1: New macOS SwiftUI project
- mode: `new_project`
- platform: `macos`
- ui: `swiftui`
- outcome: scaffold template, install Apple harness, print exact next commands

## Example 2: Adopt existing PFE companion app
- mode: `adopt_existing`
- profile: `pfe`
- outcome: no regeneration, install build harness, onboarding fragment, and release profile skeleton

## Example 3: Add tooling only
- mode: `adopt_existing`
- outcome: install local task templates and build harness without touching app source files
