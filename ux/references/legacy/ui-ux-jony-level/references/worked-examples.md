# Worked Examples

## Example 1: Greenfield spec
- request: design a new AI workspace onboarding flow
- mode: `greenfield_spec`
- outputs: `DesignBriefV1`, `ExperienceMapV1`, `FlowSpecV1`, `VisualSystemV1`

## Example 2: UX audit
- request: critique an analytics dashboard
- mode: `ux_audit`
- outputs: `DesignCritiqueV1`, prioritized fixes, revised interaction notes

## Example 3: Handoff translation
- request: translate a web concept into iOS and visionOS handoff guidance
- mode: `handoff_translation`
- outputs: `InteractionSpecV1`, `MotionAccessibilityV1`, `HandoffQAPackV1`
